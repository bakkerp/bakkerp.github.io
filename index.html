About

Ik ben een fullstack .NET developer, waarbij de focus voornamelijk ligt op de backend en databases. Ook heb ik veel ervaring met het installeren en onderhouden van programma’s (DevOps). In de projecten die ik gedaan heb, heb ik veel met klanten gecommuniceerd om een ontwerp te maken die goed afgestemd is op de klantbehoefte.

Van clean code word ik vrolijk, zeker als deze ook nog correct onder test staat. Bij problemen kan ik deze goed analyseren en snel tot de kern komen, waar ik dan graag een goede en blijvende oplossing voor bedenk. Dus ik hou niet van de quick fixes met ‘dirty programming’. Zowel zelfstandig als in teamverband (agile/scrum) kan ik goed werken.
